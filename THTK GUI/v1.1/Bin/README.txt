Графический интерфейс для программы Touhou Toolkit от xarnonymous.
Позволяет пропустить рутинные операции с командной строкой и перейти прямо к моддингу.
Для подробной информации обратитесь к файлам помощи в папке Help.
Разработка: доброанон с dobrochan.com/to
Почта: fake.dobrochan@gmail.com
Автор идеи и главный тестер: sybgamer
Репозиторий на Гитхабе: github.com/fakedobro/thtkgui

v1.1
* INI-файлы теперь хранятся в кодировке UTF. Из-за этого INI-файлы старых версий программы будут несовместимы с этой версией; однако, их можно сконвертировать в UTF вручную при помощи какого-нибудь текстового редактора.
* Версия совместима с Touhou Toolkit ver. 6, который, в свою очередь, обеспечивает совместимость с Touhou 14 (Double Dealing Character) и Touhou 14.3 (Impossible Spell Card).

A graphical interface for Touhou Toolkit made by xarnonymous.
Allows to skip routine command line operations and move on straight to modding.
For more information refer to help files in Help folder.
Developer: dobroanon from dobrochan.com/to
Mail: fake.dobrochan@gmail.com

Repository on GitHub: github.com/fakedobro/thtkgui

v1.1
* INI files are now encoded in UTF. As a result, INI files from older versions are no longer compatible, though you may convert them to UTF manually via some kind of text editor like.
* Added compatibility with Touhou Toolkit ver. 6 which in turn provides compatibility with Touhou 14 (Double Dealing Character) and Touhou 14.3 (Impossible Spell Card).
